# 🍇 칭찬 포도 (Praise Grape)

종이 "칭찬 포도 스티커"를 디지털로 옮긴 보상 차트 웹앱입니다.
아이가 잘한 일을 할 때마다 포도알을 하나씩 채우고, **30알을 다 채우면 약속한 선물**을 받습니다.

## 특징
- 📱 **설치 불필요** — `index.html` 파일만 브라우저로 열면 끝 (휴대폰/PC 모두 가능)
- 💾 **자동 저장** — 브라우저(localStorage)에 저장되어 새로고침/재방문해도 유지
- 👨‍👩‍👧‍👦 **여러 아이** — 프로필을 추가해 아이별 포도판 관리
- ✏️ **이름·목표 수정** — 이름과 "목표 선물"을 직접 눌러 편집
- 🎯 **진행률 바 + 칭찬 기록** — 몇 번째 포도알을 언제 채웠는지 기록
- 🎉 **완성 축하** — 다 채우면 색종이(컨페티)와 축하 메시지

## 사용법
1. `praise_grape/index.html` 을 브라우저에서 엽니다.
2. 칭찬할 일이 있으면 **👍 칭찬 +1** 버튼을 누르거나 포도알을 직접 탭합니다.
3. 잘못 눌렀으면 **↩︎ 취소**.
4. 포도를 다 채우면 축하 화면 → 선물 후 **🔄 새 포도**로 다시 시작.

## 데이터 저장 위치
별도 서버 없이 사용자의 브라우저 안에만 저장됩니다. (기기/브라우저를 바꾸면 데이터는 따라가지 않음)
